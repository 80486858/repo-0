# Contribution guidelines

## Introduction

This document provides information on how to contribute to the UI component library.

## Contribution process

To contribute to the UI component library, follow these steps:

1. Identify the problem you want to solve.
2. Create a bug report or change request.
3. Create a fork of the repository.
4. Follow the contribution guide and submit a pull request.
5. Make changes in a separate branch of your fork.
6. Create a pull request with your changes.
7. Go through code review and CI, build and visual regression checks.
8. After successful checks, your changes will be merged into the main branch by one of maintainers.
